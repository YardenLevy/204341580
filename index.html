Web 1.0 החברות מפרסמות תוכן שמשתמשים צורכים מנוע החיפוש איחזר תוכן ברמת המאקרו.
החיפוש היה מאוד מהיר אך פעמים רבות התוצאות לא היו מדויקות. 		התוכן כלל בעיקר תוכן סטטי, כאשר ישנה דרך אחת של פרסום תוכן ללא יצירת אינטרקציה מסוימת בין הקוראים או המפרסמים.
Web 2.0 משתמשים מפרסמים תוכן שמשתמשים אחרים צורכים, החברות בנו פלטרפורמות שאפשרו למשתמשים לפרסם תוכן למשתמשים אחרים. לדוגמה- יוטיוב, ויקיפדיה וכו'. מנועי חיפוש באינטרנט סוג 2.0 אחזרו תגיות עם תוכן ברמת המיקרו. תהליך התיוג הוא ידני, מעיק ומכסה אחוזים זניחים של www.
סוג זה מתייג דברים רבים: תמונות, אירועים, חדשות, לינקים, בלוגים וידיאו ועוד.
בנוסף, בסיס הנתונים של גוגל מאחזר גם תוכן טקסט ברמת המיקרו.		 אינטרנט סוג 2.0 עובד יותר בצורת תקשורת דו-כיוונית באמצעות רשתות חברתיות, תיוגים, בלוגים ועוד.
Web 3.0 אנשים בונים אפליקציות שמאפשרות אינטרקציה איתן, חברות בונות פלטפורמות שמאפשרות לאנשים לפרסם את שירותיהם באמצעות מינוף הקשרים בין אנשים או תכנים ייחודיים. למשל, פייסבוק. במנועי חיפוש של אינטרנט סוג 3.0 קיימת תקווה לאחזר תוכן מסוג טקסט ברמת המיקרו אשר יתוייג אוטומטית.
דבר זה מרמז על תרגום תוכן מאקרו מהסוג הראשון בכמות עצומה לכדי תוכן מסוג מיקרו.
התוצאה של כך יכולה להיות חיפוש מדויק יותר מכיוון שהתיוג יוכל לפתור חלק מאי הבהירות שהומונים וסינונים הביאו לתהליך החיפוש.		 באופן לא ברור, אינטרנט מסוג 3.0 אינו מוגדר. בינה מלאכותית והאינטרנט לומים מה המשתמש רוצה ומספקים לו חויית אינטרנט בהתאמה אישית.


Use case-Searching:

The difference between Web 1.0 to Web 3.0 in general:
Web1.0: The results of this type will be based only on the words in the query. The results probably won't be accurate and not relevant to the user.
Web3.0: The results of this type are much more user-related. They are based on semantic relations and additional actors such as other users, the user himself,
the location of the user (GPS) etc. No need for the user to deliver specific query- In web 3.0 it’s enough to send a query which is close to the user’s real intentions.

Here is a specific example for the difference:
For Web1.0
ניקח לדוגמה את השאילתה "מפלגת עלה ירוק".
אינטרנט מסוג 1.0 אשר מכיל מידע סטטי יחזיר תוצאות התואמות את מילות השאילתה במדויק. החיפוש אומנם יהיה מהיר אך לווא דווקא מדויק.
כלומר, התוצאות שיאוחזרו יהיו דפים אשר יופיעו בהם מילים כגון מפלגה שהיא לא
בהכרח זו שהתכוונו אליה, דפים עם המילה עלה או עם הצבע ירוק.
כמו כן, כמות המסמכים שיאוחזרו תהיה עצומה בכל חוסר השימוש בסמנטיקה למשל.
For: Web3.0
אינטרנט מסוג 3.0 כולל בתוכו דפים אשר מתוייגים אוטומטית. סוג זה הינו מתקדם ומכאן שהתוצאות יאוחזרו יותאמו למשתמש ולמשמעות אליה כיוון. הדפים שיאוחזרו יהיו מדויקים ככל שניתן בכל השימוש בבינה מלאכותית גם אם השאילתה נכתבה בכלליות.
כמו כן, במקרה זה קיים סיכוי שיאוחזרו דפים הקשורים לפעילות המשפלה, לפירוקה, לראשי המפלגה ולאג'נדה שלה.
